# Original / legacy JGDL games

This folder contains the unmodified legacy Java applet sources and assets:

- **Hello/** — Hello! game (Java)
- **Popcorn/** — Super Popcorn Machine (Java)
- **JGDL/** — JGDL Java Game Development Library (engine)
- **JGDL.html**, **JGDL.jar**, **.jcp**, **.jcw** — Applet/build/config
- **MyGame.java**, **MyScene.java** — Sample/template
- **src_*.txt**, **MakeJGDL.bat**, **vssver.scc** — Build/source list

The migrated browser version lives in **../migration/**.
